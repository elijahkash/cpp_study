simple_2D_lib - минималистичная, но (вроде) неплохо поддающаяся расширению 2D библиотека.
для компиляции требуется библиотека GLM

Также прикладываю небольшое приложение на SDL для демонстрации ее использования
(управление стрелками и клавишами Q,E,W,S)

Работоспособность тестировалась под  Ubuntu, gсс (10.2.0)

Исходное задание весьма расплывчатое - если глубина проработки недостаточна (а я написал это за сутки),
пожалуйста, сообщите - я расширю функциональнось и увеличу количество точек расширения.

Спасибо!
